# タスク一覧 - ゴール 1 達成への道

このディレクトリには、`todo.md`の設計書を基にした実装タスクが小分けにされています。

## 📋 タスクの進め方

**順番に実行してください。各タスクは前のタスクに依存しています。**

### ゴール 1: タスクの登録・完了・削除ができる

達成目標：

- ✅ タスク名を入力して新規登録できる
- ✅ タスクを完了状態に変更できる
- ✅ タスクを削除できる
- ✅ 登録済みタスクが一覧表示される

## 💡 ヒント

### タスク実行のコツ

1. **一つずつ確実に**: 急がず、各タスクを完了させてから次へ
2. **コミットは頻繁に**: 各タスク完了後にコミット推奨
3. **エラーは即解決**: 次のタスクに進む前に解決する

### つまずいたら

- タスクファイル内の「チェックリスト」を確認
- `task-10-testing.md`の「よくある問題と解決方法」を参照
- `todo.md`の「つまずきポイントの事前対策」セクションを確認

### 次のステップ

- ゴール 1 達成後は`must.md`を確認
- ゴール 2（サブタスク）の設計と実装に進む
- Phase 2 として世界観の追加機能を実装
