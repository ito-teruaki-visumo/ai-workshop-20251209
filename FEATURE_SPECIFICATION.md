# 機能指示書

## アプリケーション概要

**プロダクト名:** TODO アプリケーション

**説明:** ユーザー認証機能を備えた、個人用タスク管理アプリケーション。ログインしたユーザーは自分のタスクのみを表示・管理できます。

**技術スタック:**

- フロントエンド: Node.js ベース（詳細は設計フェーズで決定）
- バックエンド: Node.js
- データベース: MySQL

**対応環境:** PC ブラウザ（レスポンシブデザイン対応）

---

## 機能一覧

### 1. ユーザー認証機能

#### 1-1. ユーザー登録

- **説明:** 新規ユーザーが登録できます
- **入力:** ユーザー名、パスワード
- **出力:** 登録完了メッセージ
- **エラー処理:** 既存ユーザー名の場合はエラー表示
- **備考:** パスワードはハッシュ化して保存

#### 1-2. ログイン

- **説明:** ユーザーがログインできます
- **入力:** ユーザー名、パスワード
- **出力:** ログイン成功後、タスク一覧画面へ遷移
- **エラー処理:** ユーザー名またはパスワードが間違っている場合はエラー表示
- **備考:** セッション管理またはトークンベース認証を実装

#### 1-3. ログアウト

- **説明:** ユーザーがログアウトできます
- **出力:** ログイン画面へ遷移

---

### 2. タスク管理機能

#### 2-1. タスク一覧表示

- **説明:** ログインしたユーザー固有のタスク一覧が表示されます
- **表示項目:** タスク名、完了状態、作成日時（オプション）
- **表示順:** 作成順（降順）
- **フィルタリング:** 完了/未完了の絞り込み機能
- **検索:** タスク名の部分検索機能
- **備考:** 他のユーザーのタスクは表示されない

#### 2-2. タスク登録

- **説明:** 新しいタスクを登録できます
- **入力:** タスク名（必須）
- **出力:** タスクが一覧に追加される
- **バリデーション:** タスク名は 1 ～ 255 文字
- **初期状態:** 未完了（完了フラグ = false）

#### 2-3. タスク完了状態の変更

- **説明:** タスクの完了/未完了を切り替えられます
- **操作:** チェックボックスのクリック、または編集画面から変更
- **出力:** 完了状態が即座に反映される
- **備考:** 完了したタスクは一覧で視覚的に区別される（例：グレーアウト、取り消し線）

#### 2-4. タスク削除

- **説明:** タスクを削除できます
- **操作:** 削除ボタンのクリック
- **確認:** 削除前に確認ダイアログを表示
- **出力:** タスクが一覧から削除される
- **備考:** 削除は物理削除（ DB から完全に削除）

## 非機能要件

| 項目                 | 要件                                                           |
| -------------------- | -------------------------------------------------------------- |
| **セキュリティ**     | パスワードはハッシュ化して保存。SQL インジェクション対策を実施 |
| **パフォーマンス**   | タスク一覧の表示は 1 秒以内                                    |
| **ユーザビリティ**   | シンプルで直感的な UI。学習曲線は最小限                        |
| **アクセシビリティ** | 基本的な WCAG 2.0 準拠（設計フェーズで詳細決定）               |
| **スケーラビリティ** | 将来的にユーザー数・タスク数が増加しても対応可能な設計         |

## 機能スコープ外（将来対応予定）

- タスクの優先度管理
- タスクのカテゴリ分類
- タスクの期限設定
- タスク完了時の通知機能
- モバイル対応（現在は PC のみ）
- タスク共有機能（複数ユーザー間でのタスク共有）

## 画面イメージ

### 画面遷移図

```
ログイン画面 → (認証成功) → ダッシュボード/タスク一覧画面
  ↓
(未登録ユーザー) → ユーザー登録画面 → ログイン画面
```

### 主要画面

#### 1. ログイン画面

- ユーザー名入力欄
- パスワード入力欄
- ログインボタン
- 新規登録へのリンク

#### 2. ユーザー登録画面

- ユーザー名入力欄
- パスワード入力欄
- 登録ボタン
- ログイン画面へのリンク

#### 3. タスク一覧画面（ダッシュボード）

- ログイン中のユーザー名表示
- ログアウトボタン
- タスク新規登録フォーム
- 検索バー（タスク名検索）
- フィルタオプション（完了/未完了）
- タスク一覧
  - 各タスク行：チェックボックス、タスク名、削除ボタン
- 完了したタスク数の集計表示（オプション）

## 用語定義

| 用語               | 定義                                                 |
| ------------------ | ---------------------------------------------------- |
| **タスク**         | ユーザーが登録したやるべきこと                       |
| **完了状態**       | タスクが完了したか未完了かを示すフラグ（true/false） |
| **フィルタリング** | タスク一覧を完了状態で絞り込む機能                   |
| **検索**           | タスク名の一部で該当するタスクを絞り込む機能         |
