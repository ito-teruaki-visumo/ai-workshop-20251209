# TODO アプリ

## ゴール

- タスクの登録・完了・削除ができる

## タスク

- タスク名を入力して新規登録できる

## 設計

技術スタックは下記通りです。

- バックエンド: C# / ASP.NET Core Web API
- フロントエンド: TypeScript / Vue.js
- データベース: MySQL

## デザイン

backgroud-color: #f9f9f9
primary: #8f76d6
secondary: #ab8cff
