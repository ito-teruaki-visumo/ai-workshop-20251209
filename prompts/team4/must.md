ゴール１：タスクの登録・完了・削除ができる
タスク名を入力して新規登録できる
タスクを完了状態に変更できる
タスクを削除できる
登録済みタスクが一覧表示される

ゴール２：サブタスク
タスクに子タスク（サブタスク）を追加できる
親タスクの下に子タスクがインデント表示される
子タスクも個別に完了・削除できる
すべての子タスクが完了すると、親タスクも自動的に完了になる

ゴール３：タスク詳細
タスクに期限日を設定できる
タスクに詳細説明を追加できる
タスクに優先度を設定できる
詳細情報を表示・編集できる画面/モーダルがある
タスクをやる気にさせる何かを実装する
AI とのアイデアだしもやってみましょう

ゴール Extra：より実践的な開発
API 仕様・画面仕様のドキュメント化
単体テストの実装
gh コマンドによる PR の作成
PR の AI レビュー
