## よくある問題と解決方法

### CORS エラーが発生する

**問題**: フロントエンドから API にアクセスできない
**解決**:

- `Program.cs`の CORS 設定を確認
- ポート番号が正しいか確認（Vite は通常 5173）

### クエストが表示されない

**問題**: API 通信は成功するが画面に表示されない
**解決**:

- ブラウザの DevTools で API レスポンスを確認
- `quests.value`の値を console.log で確認

### ボタンをクリックしても反応しない

**問題**: toggle や delete が動作しない
**解決**:

- emit が正しく設定されているか確認
- 親コンポーネントでイベントハンドラーが登録されているか確認
